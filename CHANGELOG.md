# Changelog

## [1.9.4] - 2026-07-20

### Changed
- עיצוב אחיד למסכי הניהול: מערכת מבוססת Design Tokens (ממופה ל-Suite Core עם fallbacks), RTL מלא עם logical properties, מצבי focus גלויים ותמיכה ב-prefers-reduced-motion. כפתורי ה-AI בפרטי המדיה קיבלו אייקון ו-aria-label ייעודיים לכל שדה.

### Fixed
- תיוג אוטומטי: תיבות הסימון של הטקסונומיות קיבלו `name="at_ai_taxonomies[]"` — קודם "הצע תגיות" לא אסף טקסונומיות כלל.
- תיוג אוטומטי: ההצעות מסוננות כעת לאיברי taxonomy קיימים בלבד (מונע החלת מזהה שהמודל הזה); הוקשחו סניטציית הקלט ואימות `term_exists` בעת ההחלה.

### Security
- בדיקת הרשאת `edit_post` לפוסט הספציפי בפעולות הצעת/החלת התיוג; escaping (`esc_html`) לטקסט העזרה של מפתחות ה-API.

## [1.9.3] - 2026-07-20

### Fixed
- כפתור "צור טקסט חלופי AI" בפרטי המדיה: הטקסט שנוצר לא הוזן לשדה. הסיבה — הסלקטור `#attachment-details-two-column-alt-text, [data-setting="alt"]` עם `.first()` החזיר אלמנט לא-נכון במודל המדיה. כעת הערך מוזן לשדה שאליו מחובר הכפתור, עם אירוע `input` לשמירה תקינה ב-WordPress.
- הקשחת עדכון השדה מפני מאזין `change` של צד-שלישי שזורק שגיאה (למשל WooCommerce `wc-tracks` במסכי מוצר) — כדי שלא יבטל את הזנת הערך.

## [1.9.2] - 2026-07-20

### Fixed
- בדיקת עדכון מה-CDN מתבצעת במסך התוספים לכל היותר פעם ב-15 דקות, במקום להמתין ל-cache של WordPress ושל ה-manifest.

## [1.9.1] - 2026-07-20

### Fixed
- ערך מודל OpenAI ריק לא נשלח עוד ל-API; התוסף משתמש בברירת המחדל התקפה ומציג אותה במסך ההגדרות.

## [1.9.0] - 2026-07-20

### Added
- WordPress Abilities עבור יצירת alt text ותיוג taxonomies מפעולות AI.
- כפתור יצירת alt text בפרטי המדיה.
- ניתוח שדות מותאמים אישית בעת תיוג פוסט, עם סינון מפתחות רגישים.

### Changed
- תאימות ל-WordPress 7.0.2.
- OpenAI משתמש ב-Responses API ובמודל GPT-5.6 Terra כברירת מחדל.
- בחירת taxonomies כוללת כל taxonomy עם ממשק ניהול, לפי בחירת מנהל האתר.

### Fixed
- עיבוד alt text מתוזמן לאחר העלאה מחובר כעת לאירוע ה-cron.
- ניתוח תמונה עובד גם באתר מקומי או פרטי באמצעות Data URI.
- nonce ייעודי לפעולת יצירת alt text והרשאות עריכת מדיה/פוסט בכל קריאות ה-AJAX.

כל השינויים המשמעותיים בפרויקט יתועדו בקובץ זה.
