## 🎓 הדרכה מהירה

### להתחיל עם Media
```
1. מדיה → הוסף חדש
2. העלה תמונה
3. לחץ על התמונה
4. לחץ "צור תיאור AI"
5. זהו!
```

### להתחיל עם Tagging
```
1. פוסט → צור חדש
2. כתוב תוכן
3. Metabox "אופטימיזציה AI"
4. סמן: תגיות + קטגוריות
5. "הצע תגיות"
6. בחר רלוונטיות
7. "החל המלצות"
```

### להתחיל עם SEO
```
1. פוסט עם תוכן
2. "בצע אופטימיזציה"
3. קרא את 5 הקטגוריות
4. יישם את השיפורים
5. שמור ופרסם
```
