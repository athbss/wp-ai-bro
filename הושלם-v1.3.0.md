## 🚀 מה עשינו?

הוספנו **3 תכונות AI חדשות** לתוסף שלך:

### 1. 📸 כפתורים בממשק המדיה
**איפה:** בכל מקום שעורכים תמונה  
**מה:** 4 כפתורי AI - כותרת, תיאור, alt text, כיתוב  
**למה:** חוסך זמן + נגישות + SEO

### 2. 🏷️ תיוג אוטומטי חכם
**איפה:** Metabox בעורך הפוסטים  
**מה:** בוחר תגיות וקטגוריות מהקיימות  
**למה:** עקביות + דיוק + חיסכון בזמן

### 3. 🔍 אופטימיזציה SEO/AEO
**איפה:** אותו Metabox  
**מה:** 5 קטגוריות שיפור (מבנה, מילות מפתח, קריאות, snippets, AI)  
**למה:** דירוג גבוה + תשובות AI + עוד תנועה

## 🎯 איך להתחיל?

### אם התוסף מופעל (שדרוג)
```bash
# אין צורך לעשות כלום!
# התכונות החדשות כבר זמינות:

1. לך לספריית המדיה
2. פתח תמונה
3. תראה כפתורי AI חדשים!

או:

1. עורך פוסט חדש
2. חפש Metabox "אופטימיזציה AI מתקדמת"
3. התחל להשתמש!
```

### אם התוסף לא מופעל
```bash
1. הפעל את התוסף
2. הגדר API key (הגדרות → עוזר AI → מפתחות API)
3. בדוק חיבור (כפתור "בדיקת חיבור")
4. סיימנו!
```

## 🐛 בעיות? פתרונות!

### הכפתורים לא מופיעים
```
✓ רענן דף (Ctrl+F5)
✓ נקה cache
✓ בדוק שהגרסה 1.3.0
```

### ה-AI לא מציע תגיות טובות
```
✓ הרחב את התוכן (לפחות 200 מילים)
✓ צור 10-20 תגיות בסיס
✓ וודא שהתגיות באותה שפה
```

### שגיאת רשת
```
✓ בדוק API key (הגדרות → מפתחות API)
✓ לחץ "בדיקת חיבור"
✓ בדוק חיבור אינטרנט
```
