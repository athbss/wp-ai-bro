# 🔍 מערכת בדיקת תלויות - System Status

## מה חדש בגרסה 1.2.1

הוספנו מערכת מתקדמת לבדיקת תלויות שמוודאת שהתוסף יכול לפעול כראוי בסביבה שלך.

## ✨ תכונות ראשיות

### 1️⃣ עמוד System Status חדש
נגיש דרך: **AI Assistant → System Status**

מציג:
- ✅ סטטוס כל התלויות (נדרשות, מומלצות, אופציונליות)
- 📊 פרטי סביבת השרת (WordPress, PHP, זיכרון)
- 🔗 לינקים ישירים להוראות התקנה
- 💡 טיפים מעשיים

### 2️⃣ התראות אוטומטיות
- ⚠️ **התראה אדומה** - חסרות תלויות קריטיות
- 💡 **התראה כתומה** - מומלץ להתקין רכיבים נוספים
- ניתן להתעלם מהתראות על רכיבים מומלצים

### 3️⃣ בדיקות מובנות

#### נדרש (Required) ✅
- **PHP cURL** - תקשורת עם AI providers
- **PHP JSON** - עיבוד תשובות API

#### מומלץ (Recommended) 💡
- **WordPress Feature API** - ניהול מתקדם של features

## 🚀 איך להשתמש

### גישה מהירה
1. פתח **AI Assistant** בתפריט WordPress
2. לחץ על **System Status**
3. בדוק את הטבלאות למצב כל תלות

### אם יש רכיב חסר
1. לחץ על **"הוראות התקנה"** ליד הרכיב
2. העבר את המידע למנהל השרת (לרכיבי PHP)
3. או עקוב אחר ההנחיות (לספריות חיצוניות)

## 🛠️ לפי סוג משתמש

### מפתח WordPress
- גש ל-System Status לפני התחלת עבודה
- וודא שכל הרכיבים הנדרשים מותקנים
- התקן רכיבים מומלצים לחוויה מיטבית

### מנהל אתר
- בדוק System Status אחת לחודש
- העבר מידע על רכיבים חסרים לספק האחסון
- אל תתעלם מהתראות קריטיות (אדומות)

### ספק אחסון
- התקן PHP Extensions דרך:
  ```bash
  sudo apt-get install php-curl php-json
  sudo systemctl restart apache2
  ```
